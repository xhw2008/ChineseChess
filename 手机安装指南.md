# 手机安装指南

## 方法一：使用 Android Studio 构建（推荐）

### 前置条件
- 电脑上安装了 [Android Studio](https://developer.android.com/studio)
- 安卓手机，开启了开发者模式和USB调试

### 步骤

1. **打开项目**
   - 打开 Android Studio
   - 选择 "Open an existing project"
   - 选择 `/workspace` 目录

2. **等待 Gradle 同步**
   - Android Studio 会自动下载必要的依赖
   - 等待底部的进度条完成（首次可能需要5-10分钟）

3. **连接手机**
   - 使用 USB 数据线连接手机
   - 手机上允许 USB 调试

4. **构建并安装**
   - 在 Android Studio 顶部选择你的设备
   - 点击绿色的运行按钮（▶️）
   - 应用会自动安装到手机上

## 方法二：命令行构建 APK

### 步骤

1. **设置环境**
   - 确保已安装 JDK 11 或更高版本
   - 配置好 Android SDK 环境变量

2. **构建 Debug 版本**
   ```bash
   cd /workspace
   ./gradlew assembleDebug
   ```

3. **安装 APK**
   - 确保手机已连接并开启 USB 调试
   ```bash
   ./gradlew installDebug
   ```

4. **APK 文件位置**
   - 构建完成后，APK 文件位于：
   - `/workspace/app/build/outputs/apk/debug/app-debug.apk`

## 方法三：直接安装 APK

如果你已经有了 APK 文件，可以按以下步骤安装：

### 步骤

1. **传输 APK**
   - 通过 USB、蓝牙或云盘将 APK 文件传输到手机

2. **安装设置**
   - 打开手机设置
   - 进入 "安全" 或 "隐私" 设置
   - 允许安装来自未知来源的应用（临时）

3. **安装应用**
   - 在手机文件管理器中找到 APK 文件
   - 点击文件进行安装
   - 按照提示完成安装

4. **恢复设置**
   - 安装完成后，记得关闭"未知来源"选项（为了安全）

## 常见问题

### Q: 手机连接后 Android Studio 识别不到？
**A:** 
- 检查 USB 数据线是否良好
- 尝试更换 USB 接口
- 确保开发者模式已开启
- 尝试安装手机驱动程序

### Q: 安装后应用打不开？
**A:**
- 检查手机 Android 版本是否 >= 5.0 (API 21)
- 尝试卸载后重新安装

### Q: Gradle 同步失败？
**A:**
- 检查网络连接
- 配置国内镜像源（如阿里云镜像）
- 尝试使用 VPN

## 配置国内镜像源（解决网络问题）

修改 `build.gradle` 文件：

```gradle
buildscript {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/google' }
        maven { url 'https://maven.aliyun.com/repository/jcenter' }
    }
}

allprojects {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/google' }
        maven { url 'https://maven.aliyun.com/repository/jcenter' }
    }
}
```

然后重新同步 Gradle。

## 开发者模式开启方法

1. 进入手机"设置" → "关于手机"
2. 连续点击"版本号"7次
3. 返回设置，找到"开发者选项"
4. 打开"USB调试"

## 最低系统要求

- Android 5.0 (API 21) 或更高版本
- 屏幕尺寸 4.7 英寸以上（推荐）
- 4GB 内存（推荐）
